# Netflix Content Analysis using SQL

## Project Overview

This SQL project analyzes Netflix's content library to uncover trends related to movies, TV shows, countries, genres, directors, actors, and content growth over time. The project uses SQL queries to transform raw entertainment data into meaningful business insights.

## Dataset

The dataset contains information about Netflix content, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre
* Date Added
* Description

## Business Objective

* Explore Netflix's content distribution.
* Analyze movies and TV shows across countries and genres.
* Identify content trends and patterns.
* Practice real-world SQL data analysis techniques.
* Generate business insights from entertainment data.

## Business Questions Solved

### Content Analysis

* Counted the number of Movies vs TV Shows.
* Retrieved movies released in a specific year.
* Identified content added in the last 5 years.

### Country Analysis

* Found the Top 5 countries with the highest amount of Netflix content.

### Genre Analysis

* Calculated the number of titles available in each genre.
* Retrieved all Documentary content.

### Director Analysis

* Identified all content directed by Rajiv Chilaka.
* Found content records with missing director information.

### Movie & TV Show Analysis

* Found the longest movie available on Netflix.
* Identified TV Shows with more than 5 seasons.

### Actor Analysis

* Retrieved Salman Khan content released in the last 10 years.
* Identified the Top 10 actors appearing in the highest number of Indian movies.

## Key Insights

* Movies significantly outnumber TV Shows in the Netflix library.
* Content production is concentrated among a few countries.
* Certain genres dominate Netflix's catalog.
* Missing metadata can impact content discoverability.
* Long-running TV Shows contribute significantly to platform engagement.
